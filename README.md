# **Прогнозирование спроса на велопрокат BikeSouth** 

---

# 2. Цель
**Улучшить систему прогнозирования почасового спроса на велосипеды для компании BikeSouth с помощью построения гибких нелинейных моделей, превосходящих по качеству базовую линейную регрессию.**

---

# 3. Стек
* **Язык**: Python.
* **Среда**: Jupyter Notebook.
* **Библиотеки**: Scikit-learn (kNN, Decision Tree), Optuna, Pandas, NumPy.

---

# 4. Описание
**В ходе проекта выполнен исследовательский анализ (EDA) и предобработка данных о погоде и спросе на аренду. Обучены регрессионные модели kNN и Дерево решений. Подбор гиперпараметров автоматизирован через Optuna с использованием оценки по метрике RMSE на 5 фолдах кросс-валидации.**

---

# 5. Результаты
* Оптимизированы нелинейные модели, которые успешно превзошли базовую линейную регрессию заказчика.
* Проведена и сохранена финальная, лучшая модели.

**Итоговые метрики модели:** 
| Модель | RMSE | R² |
| :--- | :--- | :--- | 
| **Линейная регрессия (Baseline)** | *411.5* | *0.58* |
| **kNN (Optuna)** | *313.7* | *0.76* |
